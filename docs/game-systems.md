# 게임 시스템

## 전투

- **데미지**: `(ATK × skillMult - DEF × 0.5) × critMult`, 최소 1
- **버프**: attack_up/defense_up/shield 실적용 (데미지 계산에 반영)
- **상태이상**: poison/burn/bleed(DOT), stun(행동불가), regen(HP회복)
- **웨이브**: 던전 3웨이브 자동 진행, 웨이브별 보상
- **속도**: 적 행동 순서 speed 기반 정렬
- **MP 회복**: 3%/턴 + 특성/유물 보너스
- **자동 전투**: A키 토글, 500ms/speed 간격, 광역기 우선, 자동 포션(HP<30%)
- **전투 통계**: 총데미지/받은피해/최고크리/턴수/스킬수

## 장비 (10부위)

weapon / offhand / helm / shoulders / chest / gloves / belt / legs / boots / accessory

- **등급**: common → uncommon → rare → epic → legendary
- **클래스 제한**: requiredClass 필드, ID 접두어 (dk_/sm_/hn_/pr_/as_)
- **자동 강화**: 중복 드랍 시 addItemSmart → 강화 경험치
- **강화 비용**: +1: 2개, +2: 2개, +3: 4개... `2^max(1,level)`
- **골드 강화**: `등급기본가 × level²`, 성공률 `max(5, 50-(level-5)×1.5)`%
- **보석 소켓**: common/uncommon 1소켓, rare 2, epic/legendary 3
- **세트 보너스**: 7세트, 2/3/5세트 단계별 패시브+액티브

### 세트 액티브 효과
- reflect: 피격 시 확률 피해 반사
- lifesteal_on_crit: 크리 시 HP 흡수
- bonus_damage: 공격 시 확률 추가 피해
- hp_regen_per_turn: 턴당 HP% 회복
- mp_regen_per_turn: 턴당 MP% 추가 회복

## 던전

| 던전 | 레벨 | 보스 |
|------|------|------|
| 버려진 지하묘지 ~ 살게라스의 무덤 | 1~60 | Lv15부터 보스 등장 |

### 무한던전 (심연의 나락)
- 0~999층, Lv60 진입
- 스케일링: `1 + 층 × 0.5`
- 10층마다 보스 (전체 보스 풀 랜덤)
- 사망 시 -10층
- 드랍: 60% 고정 (common~epic 층별 가중치) + 레전드리 독립 `0.1% + 층×0.028%`
- 보스 100% 레전드리

### 주간 보스
- 주 1회 도전, 보스 3배 스탯
- 보상: 50,000G / 30,000 EXP / 레전드리 / 젬 10개

## 성장 시스템

### 스킬 레벨업
- 골드 비용: `1000 × (level+1)²`, 최대 Lv20
- 효과: 데미지 배율 +5%/레벨

### 특성 트리
- 3갈래: 공격(ATK/크리율/크리뎀), 방어(HP/DEF/재생), 유틸(MP/MP재생/골드)
- 레벨당 1포인트, 리셋 10,000G

### 환생 (Prestige)
- Lv60+ 시 리셋 (레벨/스킬/특성/심연진행)
- 유지: 장비/강화/업적/골드/젬/유물
- 누적 보너스: 스탯+2%/경험치+10%/골드+5%/드랍+1% per 환생
- 젬 보상: `50×환생Lv + (현재Lv-60)×2 + 최고층×0.5`

### 유물
- 7종, 젬으로 Lv1~10
- 경험치서(+10%)/재물서(+10%)/행운부적(+5%)/생명결정(+5%)/마나결정(+5%)/파괴룬(+3%)/수호룬(+3%)
- 환생 유지

## 경제

### 골드 사용처
- 포션/장비 구매, 골드 강화, 스킬 레벨업, 특성 리셋

### 젬 사용처
- 유물 업그레이드, 펫 소환, 보석 장착

### 상점
- 포션: 무한 구매, 고정가
- 장비: 6칸 랜덤, 2시간 갱신, 클래스 전용
- 제작: 재료 → 포션/젬

### 일일 보상
- 요일별 500~10,000G, 중복 수령 불가

## 수집

- **업적**: 17개 (처치/레벨/강화/심연/골드/도감), 골드/젬 보상

- **도감**: 몬스터 처치 기록 + epic/legendary 드랍 히스토리
