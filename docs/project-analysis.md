## 3. 핵심 시스템

### 전투
- 3웨이브 자동 진행, 웨이브별 보상
- 상태이상: poison/burn/bleed/stun/regen/defense_up/attack_up/shield (실적용)
- 속도 기반 적 행동 순서
- MP 자연 회복 3%/턴 (+ 특성/유물 보너스)
- 자동 전투 (A키, 광역기 우선, 자동 포션 HP<30%)
- 전투 속도 x1/x2/x3
- 전투 통계 (총데미지/받은피해/최고크리/턴수/스킬수)

### 장비
- 10부위: weapon/offhand/helm/shoulders/chest/gloves/belt/legs/boots/accessory
- 자동 강화: 중복 장비 → 강화 경험치 자동 전환
- 골드 강화: 성공률 감소형 (+5까지 50%, +35+ 5%)
- 보석 소켓: 등급별 1~3소켓, 10종 보석
- 7개 세트: 패시브(스탯%) + 액티브(반사/흡혈/추가피해/턴회복)
- 직업별 장비 제한 (requiredClass)

### 무한던전 (심연의 나락)
- 0~999층, 선형 스케일링 `1 + 층 × 0.5`
- 10층마다 보스, 사망 -10층
- 드랍: 60% 고정 (common~epic 층별 가중치) + 레전드리 독립 (0.1% + 층×0.028%)
- 보스 100% 레전드리

### 환생 (Prestige)
- Lv60+ 시 리셋
- 영구 보너스: 스탯+2%/경험치+10%/골드+5%/드랍+1% per 환생
- 젬 보상: `50×환생Lv + (현재Lv-60)×2 + 최고층×0.5`
- 유지: 장비, 강화, 업적, 골드, 젬, 유물

### 유물
- 7종, 젬으로 레벨업 (최대 Lv10)
- 경험치/골드/드랍률/HP/MP/ATK/DEF 영구 % 보너스
- 환생해도 유지

## 5. 보너스 적용 순서 (전투 스탯)

```
기본 (레벨 스케일링)
  → + 장비 스탯 (강화 배율 적용)
  → + 보석 소켓 스탯
  → × 세트 보너스 %
  → × 환생 보너스 (2%/환생)
  → × 특성 보너스
  → × 칭호 보너스
  → × 펫 보너스
  → × 유물 보너스
```
